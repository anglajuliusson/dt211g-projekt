(async) initMap() → {Promise.<void>}

Initierar Google Maps-kartan och markerar användarens aktuella position på kartan. Hämtar användarens geografiska position (latitude och longitude) och använder dessa för att centrera kartan och placera en markör vid användarens position. Om det inte går att hämta användarens position eller om det sker ett annat fel vid inladdning av kartan, loggas ett felmeddelande i konsolen.
